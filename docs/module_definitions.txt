MODULE LEVEL DEFINITIONS
v0.1
format of function signature --> return_type name_of_functions(type argument)

budget_track module:

void run() -- main function that will run.

---------------------------------------------------------------------

cmd_user_interface module:

command line pattern literal -- for use by docopt module.
dict get_parsed_user_input() -- returns a parsed user input.

---------------------------------------------------------------------

request_processor module:

void process_request(dict user_input) -- takes user input to perform
a budget-related request.

void _create_budget(string budget_name, string reset_period_type,
	                bool move_leftover) -- creates a brand new budget
                    and stores it as a file.

void _remove_budget(string budget_name) -- removes a given budget.
void _remove_topic(string budget_name, string topic_name) -- removes a given topic.

void _add_new_topic(string budget_name, string topic_name,
float limit) -- creates a topic for a particular budget.

void _change_reset_period(string period_type)

void _increment_current_spending(string budget_name, string topic_name,
float amount) -- update current spending for a given topic and budget.

void _list_budgets_topics_and_expenditures(string budget_name) -- lists
                                           budget's topics and their
                                           current spending.
